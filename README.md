# milliwatson
The newest pint-sized trivia super star

# To set up:
Linux:
```bash
sudo apt-get install tesseract -y
pip3 install -r requirements.txt
```

MacOS:
```bash
brew install tesseract
pip3 install -r requirements.txt
```
